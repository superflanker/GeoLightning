Module contents

GeoLightning Package

Summary

This package contains modules, optimizers and tools for spatio-temporal localization of atmospheric lightning events based on Time-of-Arrival (TOA) data.

The algorithms include original implementations and custom wrappers for metaheuristic solvers (e.g., PSO, GA, ESO) integrated with the STELA method.

Author

Augusto Mathias Adams <augusto.adams@ufpr.br>

Submodules

  • Solvers: Optimizers and wrappers

  • Stela: Core algorithms for likelihood computation

  • Utils: Constants and distance functions

Notes

This package is part of the activities of the discipline EELT 7019 - Applied Artificial Intelligence, Federal University of Paraná (UFPR), Brazil.
